Object Literals

Objects in JavaScript have properties and things they can do (aka methods).
For instance a blog object could have as props: title, content and author and as methods: publish, unpublish and delete
Objects are used to create data structures

Primitive types

  • numbers
  • strings
  • Booleans
  • null
  • undefined
  • symbols

Reference types

  • all types of objects
  • object literals
  • arrays
  • functions
  • dates
  • all other objects

When we create a primitive value and we store it into a variable that value is stored into something called the stack. It is really fast to access but there is not that much memory.
When we create something from the reference variables type they are stored in the heap. More space available but a bit slower.
